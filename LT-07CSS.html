This is a sample article used to demonstrate how a sticky header works. As you scroll down the page, the header will remain at the top so that navigation is always available.
Sticky headers are commonly used in blogs and websites to improve user experience. They allow visitors to easily access menu options without scrolling all the way back up.
This paragraph adds more content for scrolling. When a page contains a lot of text, a sticky header helps users explore the site faster and with less effort.
You can use position: sticky and top: 0 in CSS to create this effect. It is simpler than using JavaScript and works smoothly on modern browsers.
